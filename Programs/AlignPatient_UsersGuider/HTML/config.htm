Конфигурационные файлы

 Top  Previous 

 

 

Имеется два конфигурационных файла, которые  располагаются в тех же директориях, где каждая из выполняемых

программ. Они называются - DoctorDisplay.exe.config, PatientDisplay.exe.config. Оба имеют одни и те же поля.

 

PatientDisplay.exe.config задаёт параметры, которые загружаются при старте программы PatientDisplay.exe

Основные из них:

•RoiX, RoiY, RoiWidth - задают область экрана, которая будет увеличена (zooming) во весь экран.

    Если RoiWidth < 50 - zooming отключен.

•Threshold - задаёт порог сравнения текущего видеокадара и реперного снимка.
