1. При запуске программы DoctorDisplay не появляется видео изображение.

 

               Проверьте:

• питание нетбука (светодиодный индикатор), разъём внешнего питания.

• перезагрузите нетбук.

• сетевой разъём нетбука

 

2. Нет видео изображения на экране нетбука, нетбук не загружается при включении питания

 

          Проверьте:

• питание нетбука (светодиодный индикатор), разъём внешнего питания.

• перезагрузите нетбук.

• сетевой разъём нетбука

 

3. Ошибка при спасении реперного снимка в базу данных.

4. Ошибка при загрузке реперного снимка из базы данных.
